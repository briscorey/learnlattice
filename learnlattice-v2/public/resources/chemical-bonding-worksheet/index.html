Electron Configuration Reminder

Atoms want to achieve a full outer shell (like noble gases). They do this by:

  • Losing electrons ‚Üí become positive ions (cations)
  • Gaining electrons ‚Üí become negative ions (anions)
  • Sharing electrons ‚Üí form covalent bonds

Part 2 ‚Äî Covalent Bonding

In covalent bonds, pairs of electrons are shared between atoms. Each shared pair counts as one bond.

Worked Example: Water (H‚ÇÇO)

Oxygen has 6 outer electrons (needs 2 more). Each hydrogen has 1 outer electron (needs 1 more). Oxygen shares one pair with each hydrogen atom. Result: O has 8 electrons in outer shell ‚úì, each H has 2 ‚úì

Word Equations ‚Üí Symbol Equations

Rules:

  1. Write correct formulas for all reactants and products
  2. Balance the equation (same number of each atom on both sides)
  3. Add state symbols: (s) solid, (l) liquid, (g) gas, (aq) dissolved in water

When ionic compounds dissolve in water, they split into separate ions. We can write ionic equations that only show the ions that actually change (the spectator ions cancel out).
